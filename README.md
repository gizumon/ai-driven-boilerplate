# AI-driven development sample

このリポジトリは、AI駆動開発のベストプラクティスを実践するためのサンプルリポジトリです。

## Getting Started

このサンプルリポジトリでは既にサンプルとしてSubAgent、Skills、Rulesが定義されていますが、実際のプロジェクトの内容に応じて下記の手順で再生成行ってください。

1. 開発プランを `plans` ディレクトリに定義する
2. Claudeにプランに基づいてSubAgent、Skills、Rulesを生成するように依頼する
  - CLAUDE.mdに記載しているAI-Driven Development運用モデルがコアとなり、SubAgent、Skills、Rulesが適切に生成され管理されることを期待。
